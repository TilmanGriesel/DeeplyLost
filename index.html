DeeplyLost

Download View on GitHub

You lost the island where you came from or trapped in a bug?
DeeplyLost will bring you back home.

How To

First of all, this should not be a cheat tool. So I decided that it is necessary to place a camp fire somewhere on the island you want to teleport to. This camp fire is used as a reference for the re-spawn. Keep in mind that you can't travel to islands without a campfire.

Basics
  • Extract the lates release zip
  • Start DeeplyLost.exe
  • Navigate to your Stranded Deep save game called Save.json*
  • Drag the Save.json into the DeeplyLost window
  • Select the island you wan't to go
  • Press the Teleport button
  • Start the game and press Load Game
Advanced
  • Double click the island name label to rename it
*Steam location: SteamLibrary\steamapps\common\Stranded Deep\Stranded_Deep_x64_Data\Data
